# 文明6 · 高校娱乐赛 乔尔赛制 BP工具

## 📖 项目简介

这是一个为《文明6》高校娱乐赛设计的**BP（禁用/选择）流程管理工具**，支持**随机模式**和**自选模式**双模式切换，并附带独立的**战术规划面板**用于赛前战术准备。

### 核心功能

| 功能 | 说明 |
|------|------|
| 🗺️ 地图抽取/选择 | 从15张预置地图中随机抽取或手动选择 |
| 🔴🔵 队伍分配 | 红蓝两队分配A/B位置（1-6选人次序） |
| 👑 领袖池抽取 | 从6大领袖池中随机抽取2个或手动选择 |
| 🚫 两轮BAN选 | A队先BAN → B队后BAN，BAN数可自定义 |
| ⚔️ 选人环节 | 按1(A)→2(B)→3(B)→4(A)→5(A)→6(B)顺序选人 |
| ⏱️ 15分钟计时器 | 选人阶段倒计时 |
| 📋 战术规划面板 | 独立的赛前分析工具（见下方） |

### 战术规划面板

通过启动画面可跳转到独立的战术规划面板，提供：

| 功能 | 说明 |
|------|------|
| 🎨 地图标记板 | 画笔/橡皮/文字工具，在地图上直接标注 |
| 📍 点位评分 | A1-A5点位1-5星评分 |
| 🏛️ 城邦速览 | 查看本局城邦，悬停显示类型和加成效果 |
| 👑 领袖分类 | 预设/自定义分类（自爆位、海战位等），支持拖拽 |
| 📝 战术笔记 | 按地图独立保存的战术笔记 |
| 💾 本地存储 | 所有数据保存在浏览器本地 |

### 环境要求

- 任意现代浏览器（Chrome / Edge / Firefox）
- 本地HTTP服务器（二选一）：

**方式一：使用Python**
```bash

python -m http.server 8080
```

**方式二：使用VS Code**
安装 `Live Server` 插件，右键 `index.html` → `Open with Live Server`

**方式三：使用提供的批处理脚本（Windows）**
双击 `start.bat`

## 📖 用户手册

### 一、主流程操作指南

#### 1. 启动界面

打开页面后看到启动画面，有两个入口：

- **🎲 随机模式 / ✋ 自选模式** — 顶部切换开关
- **📋 战术规划面板** — 跳转到独立分析工具
- **点击开始** — 进入BP主流程

#### 2. 步骤1：选择地图

| 模式 | 操作 |
|------|------|
| 随机模式 | 点击「🎲 抽取地图」随机抽取 |
| 自选模式 | 从地图网格中点击选择，实时预览缩略图 |

#### 3. 步骤2：队伍分配

| 模式 | 操作 |
|------|------|
| 随机模式 | 点击「🎲 随机分配」随机分配红蓝队到A/B队 |
| 自选模式 | 手动指定红队对应A队还是B队，蓝队自动对位 |

**队伍说明：**
- **A队**：选人次序为 第1、第4、第5 位
- **B队**：选人次序为 第2、第3、第6 位

#### 4. 步骤3：领袖池

| 模式 | 操作 |
|------|------|
| 随机模式 | 点击「🎲 抽取两个领袖池」 |
| 自选模式 | 从6大池子中点击勾选2个，确认后生效 |

鼠标悬停领袖名称可查看文明能力和领袖能力详情。

#### 5. 步骤4：BAN选（两轮制）

**BAN数说明：**
- 随机模式：自动随机生成2-6之间的BAN数
- 自选模式：可用 +/- 按钮手动调节BAN数（1-6）

**阶段1：A队BAN**
1. 在左侧池1和池2中勾选要BAN的领袖
2. 达到指定BAN数后，「确认A队BAN」按钮激活
3. 点击确认进入下一阶段

**阶段2：B队BAN**
1. 在全部领袖中勾选要BAN的领袖（可看到A队已BAN的）
2. 达到指定BAN数后，「确认B队BAN」按钮激活
3. 点击确认完成BAN选

#### 6. 步骤5：地图详情 & 选人

**地图查看：**
- 点击A1-A5点位查看魅力/大陆/资源预览图
- 切换总览图/点位图
- 点击预览图可放大查看

**选人规则：**
- 顺序严格按 1(A)→2(B)→3(B)→4(A)→5(A)→6(B)
- 左侧为A队可选池（红标为B队已BAN）
- 右侧为B队可选池（蓝标为A队已BAN）
- 中间显示6个选人槽位，当前轮到谁选会高亮

**计时器：**
- 点击「开始计时」启动15分钟倒计时
- 剩2分钟时数字变红闪烁
- 超时弹出提示

**选人完毕：**
- 6人选满后显示结果
- 可点击「重置选人」重新选择

### 二、战术规划面板操作指南

通过启动画面的「📋 战术规划面板」进入（新标签页打开）。

#### 1. 选择地图
顶部下拉菜单选择要分析的地图（与主流程独立）。

#### 2. 地图标记板
| 工具 | 操作 |
|------|------|
| ✏️ 画笔 | 选中后在地图上按住鼠标拖动画线 |
| 🧹 橡皮 | 擦除已画线条 |
| 🔤 文字 | 点击地图位置后输入文字标注 |
| 颜色选择 | 6种颜色可选 |
| 线宽 | 1-8档可调 |
| ↩️ 撤销 | 撤销上一步操作 |
| 🗑️ 清空 | 清除所有标记 |

**注意：** 地图标记按地图ID保存在浏览器中，切换地图不会丢失。

#### 3. 点位分析
- 点击A1-A5点位查看详细预览图
- 拖动滑块给点位打分（1-5星）
- 评分自动保存

#### 4. 本局城邦
- 显示当前地图绑定的城邦列表
- 城邦按类型（工业/军事/科技/贸易/文化/宗教）显示彩色标签
- **鼠标悬停城邦名称**查看城邦类型和完整加成效果

#### 5. 领袖分类
| 操作 | 方法 |
|------|------|
| 查看分类 | 点击分类标签切换 |
| 加入分类 | 在「全部领袖」视图中点击领袖，或拖拽到分类区 |
| 移出分类 | 在分类区点击已放入的领袖（会变红） |
| 新建分类 | 点击「+ 新建分类」，输入名称 |
| 删除分类 | 点击分类标签上的 × |

预设分类：`自爆位` / `海战位` / `打钱位` / `发育位`

#### 6. 战术笔记
- 自由记录战术思路
- 点击「💾 保存全部」保存所有数据
- 数据按地图独立存储

### 三、数据存储说明

所有战术数据存储在浏览器的 `localStorage` 中：

| 数据类型 | 存储键 |
|------|------|
| 地图标记、评分、笔记、领袖分类 | `tactics_data_{地图ID}` |

**注意：**
- 数据仅保存在当前浏览器中
- 清除浏览器缓存会导致数据丢失
- 不同地图的数据完全独立

## 📝 自定义配置

### 添加新地图

在 `index.html` 和 `tactics.html` 的 `MAP_DATA` 对象中添加：

```javascript
16: {
    name: '新地图名称',
    mapUrl: './maps/16_新地图/地图.png',
    pointUrl: './maps/16_新地图/点位.png',
    A1: { charm: '...', continent: '...', resource: '...' },
    // ... A2-A5 同上
    cityStates: ['城邦1', '城邦2', ...]
}
```

### 修改默认BAN数

在 `randomPools()` 方法中修改随机范围：
```javascript
this.banCount = Math.floor(Math.random() * 5) + 2; // 当前：2-6
// 改为固定3个：this.banCount = 3;
```
